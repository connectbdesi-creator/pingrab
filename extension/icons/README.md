# Extension icons

Place 4 PNG icons here before loading the extension:

- `icon16.png` — 16×16
- `icon32.png` — 32×32
- `icon48.png` — 48×48
- `icon128.png` — 128×128 (used on the Chrome Web Store listing)

## Quick way to generate

1. Create or export your square logo as a 512×512 PNG.
2. Use https://realfavicongenerator.net or https://icon.kitchen — upload the 512×512 and download the 16/32/48/128 set.
3. Rename to match the filenames above and drop them in this folder.

## Design guideline

- Red background `#e60023` (matches brand)
- White "P" glyph or a download arrow
- Rounded square or circle; keep content inside the inner 80% to survive Chrome's safe area.
